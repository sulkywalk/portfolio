
# React Portfolio

A personal portfolio website built using React and Vite, featuring separate sections for Projects, Blog, Skills, Contact, and more. This portfolio is designed to be fast, modern, and easy to customize.

---

## Features
- Light / Dark theme toggle (saved to localStorage)
- Client-side routing for blog posts
- Skills and Projects sections
- Projects modal with demos & GitHub links
- Responsive layout and simple animations
-  Built with Vite for fast development

## Installation and Setup

Clone the repository:

```bash
git clone https://github.com/YOUR-USERNAME/react-portfolio.git
cd react-portfolio
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
